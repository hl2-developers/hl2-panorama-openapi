## Description

The [OpenAPI](https://openapis.org) specification for hl2 Panorama API.

## Scripts

- `npm install`: install dependencies
- `npm run validate`: validate [OpenAPI](https://openapis.org) specification
- `npm run editor`: start swagger editor
